# 11. Glossary

| Term | Definition |
|---|---|
| Synchrotron | A particle accelerator that produces extremely bright X-rays for scientific imaging and analysis. |
| Beamline | A dedicated experimental station fed by the synchrotron beam. A large facility has many beamlines. |
| Hutch | A lead-shielded enclosure where samples are irradiated. Humans must exit before beam can be opened. |
| Interlock | A safety system preventing the beam shutter from opening unless all hutch doors are secured. |
| Beam Dump | Unplanned loss of beam current. All scans pause until the ring is refilled. |
| Ring Stability | In-game metric representing how reliably the storage ring delivers beam. Decays with age and hazards. |
| Brilliance | The technical measure of X-ray beam quality — the game name "The Brilliant Facility" references this. |
| DLSR | Diffraction-Limited Storage Ring — the current generation of ultra-bright synchrotrons. |
| Committed Proposal | A proposal selected in the Proposal Review screen. Carries a rep penalty if 0 samples are done. |
| Rapid Review | A bonus proposal appearing during live gameplay. No penalty; optional; expires in 40s. |
| Batch | The set of N samples in a single user proposal. Partial completion is always rewarded. |
| Publication Delay | Planned mechanic: rep from completed experiments arrives 1–3 cycles later after the paper publishes. |
