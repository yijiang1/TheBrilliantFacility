# 5. Hazards

## 5.1 Beam Stop *(Implemented)*

The primary hazard in the M1 prototype.

| Property | Detail |
|---|---|
| Trigger | Single roll at cycle start; chance = (100 − ring_stability) % |
| Duration | 0.5–1.0 game days in cycle 1; maximum grows +0.5 real seconds per subsequent cycle |
| Effect | All scan slots freeze. Prep Table continues to work. Player can still move and collect. |
| Ring cost | None — ring stability is fixed for the entire cycle |
| Visual | Slim red banner below HUD. Scan progress bars pause. Beam path graphic turns off. |
| Recovery | Scan timers resume exactly where frozen. "Beam restored" flash message. |

## 5.3 Ring Stability

Ring Stability is a global HUD bar (0–100%). It is **fixed at its base value for the entire active cycle** — it does not decay during gameplay. Starts at **98%** on the first cycle of the first year. Drops **2 points per cycle**, flooring at **30%**.

| Stability Range | Beam Stop Risk | Tier 2 Hazard Rate | Other Effects |
|---|---|---|---|
| 100–80% | Normal | Rare | No special effects |
| 80–60% | Elevated | Moderate | Beam dump duration +20% |
| 60–40% | High | Frequent | Scan time +10% |
| 40–20% | Very High | Very Frequent | Tier 3 event guaranteed before cycle end |
| < 20% | Constant | Constant | Director visit guaranteed; funding grant −25% |

> Ring Maintenance upgrade effect is pending redesign for the new per-cycle decay model.
