# 1. Game Overview

## 1.1 Concept

The Brilliant Facility is a top-down arcade management game in which the player runs an entire X-ray synchrotron research facility. The player controls a Facility Director — simultaneously scientist and manager — racing between laboratory stations, collecting and processing samples for user experiments, managing cascading hazards, and making strategic decisions about which proposals to commit to each beamtime cycle.

Each playthrough spans multiple years. A year contains three 90-day beamtime cycles. Each cycle is a **3-minute real-time action phase**. Between cycles, the player buys upgrades. At year end, new funding is awarded based on reputation earned.

## 1.2 Design Pillars

- **BRILLIANT** — Every mechanic is grounded in real synchrotron science.
- **FRANTIC** — The clock is always ticking. Beamtime is sacred and finite.
- **STRATEGIC** — Pre-cycle proposal selection matters as much as execution.
- **GROWING** — The facility evolves from a 1990s machine to a world-class source.
- **REWARDING** — Every sample counted, every paper published, every upgrade earned.
