# 8. HUD & User Interface

## 8.1 In-Cycle HUD

| Element | Detail |
|---|---|
| Cycle Timer | Top-centre. Shows "XX days left" counting down from 90 to 0. Turns amber <20 days, red <15 days. |
| Cycle / Year Label | Below timer. Shows "CYCLE N / YEAR N". Updates each cycle. |
| Samples Counter | Top area. Running count of total samples characterised this cycle. |
| Reputation | Top area. Live total with animated +/− pop-ups. |
| Inventory Counter | Top area. Shows "🧪 N/3 held" — how many samples the player is carrying. |
| Ring Stability | Top-right. Colour bar: green (>60%) → amber → red (<30%). |
| Active Jobs Panel | Right column (180px). All active job slots with live state labels and pip bars. Rapid Review proposals share this panel, appearing in any empty slot. |
| Beam Stop Banner | Slim red strip below HUD. Shows "⚡ BEAM STOP" + countdown. Disappears on recovery. |
| Status Flash | Bottom-centre. Context messages (action feedback, warnings, completions). |
| Prompt Text | Bottom-centre. Context-sensitive [Space] prompt based on current proximity. |

## 8.2 Cycle End Screen

| Element | Detail |
|---|---|
| Results summary | Samples done, reputation total, ring stability % |
| Cycle Analytics | Auto-calculated from real play: proposals done / partial / failed · throughput (s/sample) · rep/sample |
| Paper outcome | Text banner indicating likely publication tier based on rep earned |
| Funding banner | New funding awarded (year-end cycles only) |
| Upgrade shop | Right column — 6 upgrades, greyed out if unaffordable |
| Export Stats JSON | Dev button (bottom-left) — downloads all `localStorage` stat records as a `.json` file |
| Clear Stats | Dev button (bottom-left) — wipes the `localStorage` stat history |

## 8.3 Controls

| Input | Action |
|---|---|
| WASD / Arrow Keys | Move the Facility Director character |
| [Space] | Context-sensitive: collect sample from NPC / deposit sample at station / pick up station output |
| [Space] held at Interlock | Performs the interlock procedure (1.8s hold while stationary with prepped sample) |
| Click | Accept a proposal from the Rapid Review board or the Proposal Review screen |
| `` ` `` (backtick) | Toggle Layout Mode (dev tool) |

## 8.4 Layout Mode (Dev Tool)

Press **`` ` ``** during gameplay to enter Layout Mode. A banner appears at the bottom of the screen confirming the mode is active.

**What it does:**
- Makes HUD elements draggable so you can visually reposition them (timer, cycle label, score, pause/speed buttons, phase text, job header).
- Shows the **Board Layout sliders** panel (top-left) for adjusting ring radius, prep radius, and beamline rotation offset.
- Dragging an element logs its new position to the console on release.

**Exiting:**
- Press **`` ` ``** again to exit. All elements lock back in place and final positions are logged to the console as a summary you can copy into code.
